Community

Yetibot is primarily built by @devth plus contributions from a number of other users.

Slack

Join the party on Yetibot Slack.

Primary channels are:

  • #general - user discussion, play with Yetibot, ask questions
  • #dev - dev chat, suggest features, discuss the inner workings of Yetibot

Feel free to ping @devth with questions.

Discussion

There's a dedicated [community GitHub repo](https://github.com/yetibot/community) for discussions. Open a new issue to discuss questions, suggestions, comments, etc.

About Yetibot

Yetibot was started in 2011 by @devth as a fun way to learn Clojure.

It has since evolved into a powerful expressive chat bot used by teams for automation and for fun.
